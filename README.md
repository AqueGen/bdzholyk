# Бджолик — демо-сайт

Сучасний односторінковий редизайн сайту БО БФ «ВІРИМО – ВСЕ БУДЕ ДОБРЕ» (м. Суми).

**Демо:** https://aquegen.github.io/bdzholyk/

## Стек

- HTML5 (semantic, `lang="uk"`)
- Tailwind CSS (через Play CDN)
- Google Fonts: Comfortaa + Inter
- Vanilla JS (без фреймворків)
- Без білду — просто відкрий `index.html`

## Структура

```
index.html       — головна сторінка (всі секції)
assets/
  favicon.svg    — фавіконка-бджолик
  mascot.svg     — маскот-бджолик для hero
docs/superpowers/specs/
                 — дизайн-специфікація
```

## Секції

1. Hero — заголовок, слоган, маскот, CTA
2. Про нас — маніфест + лічильники
3. Напрямки — 6 карток діяльності
4. Галерея — фото подій (плейсхолдери з picsum.photos)
5. Як долучитися — волонтер / донат / партнер
6. FAQ — accordion
7. Контакти / футер

## Адаптивність

Mobile-first. Брейкпоінти Tailwind: `sm` 640px, `md` 768px, `lg` 1024px, `xl` 1280px.
Без горизонтального скролу від 320px.

## Мови

Українська (за замовчуванням) + English. Перемикач `UK / EN` у шапці.
Вибір зберігається в `localStorage`, також підтримує `?lang=en` у URL.
Всі тексти у словнику `I18N` всередині `index.html` (без зовнішніх файлів).

## Деплой

GitHub Pages з `main` branch, root.

## Зміст

Збережено всю суть оригінального сайту: цільову аудиторію, мову, маскот,
шість напрямків, гасло «Від маленької дії — до великої мрії». Покращено
дизайн, типографіку, структуру, accessibility.

Фото в галереї та картках — плейсхолдери з [picsum.photos](https://picsum.photos),
готові до заміни на реальні зображення фонду.
